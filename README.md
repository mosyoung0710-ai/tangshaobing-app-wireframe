# 糖哨兵 App 新增功能线框原型

线上预览：https://mosyoung0710-ai.github.io/tangshaobing-app-wireframe/

这是用于产品评审和开发对齐的静态线框原型。请优先通过上方 GitHub Pages 链接查看最新发布版本。

本版覆盖页面：

1. 页面流转图
2. 基础信息
3. 绑定阶段：血糖单位选择
4. 模式设置与预览改版
5. 警报小睡卡片
6. 我的
7. 设备详情
8. 亲情连接 / 选择照护对象 / 更换照护对象
9. 修改基础信息
10. 修改血糖单位
11. 账号与安全
12. 多语言
13. 英文文案清单
14. 弹窗与异常提示

设计口径：

- 只做线框结构和页面排版，不追求最终 UI 视觉。
- 页面流转图用于快速理解主要入口和跳转关系，点击带截图的卡片可跳到对应原型页。
- 页面右侧说明只写图里表达不完整的规则、状态、联动和异常。
- 英文文案清单按用户已确认的本地表格口径同步。
- 警报小睡卡片覆盖展开、收起、小睡倒计时、小睡中详情、小睡功能未开启等状态。
- 不包含真实用户信息、接口地址或患者数据。

本地维护说明见 `LOCAL_DEV.md`。
